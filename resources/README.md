# Resources

Curated collection of learning resources, bookmarks, and references.

## Contents

- **bookmarks.md** - Useful websites, tools, and online resources
- **courses.md** - Online courses, tutorials, and learning paths
- **articles.md** - Technical articles and blog posts to read
- **books.md** - Technical books and reading list
- **tools.md** - Development tools, utilities, and software

## Organization

Resources are categorized by:
- Technology/domain
- Priority (must-read, should-read, nice-to-read)
- Status (read, in-progress, to-read)

## Contributing to Resources

When adding new resources:
1. Add to appropriate category file
2. Include brief description
3. Note why it's relevant
4. Tag with relevant technologies
5. Mark priority/status
